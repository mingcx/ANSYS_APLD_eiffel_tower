# ANSYS_APLD_eiffel_tower

This is Ansys Parametric Design Language code to generate Eiffel tower and apply displacement boundary conditions. This model is for visual illustration only.
How to use:
1. Copy the code from https://github.com/mingcx/ANSYS_APLD_eiffel_tower/blob/main/APDL%20code to ANSYS APDL command line and presse Enter
2. Save https://github.com/mingcx/ANSYS_APLD_eiffel_tower/blob/main/APDL%20code as .txt and read it use ANSYS APDL GUI
